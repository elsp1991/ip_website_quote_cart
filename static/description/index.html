Overview

Website Quote Cart adds a separate quotation cart to your Odoo eCommerce store, allowing customers to request quotes for products instead of purchasing directly. Perfect for B2B businesses, custom pricing scenarios, or products that require negotiation.

How It Works

  1. Add to Quote - Customers click the "Add to Quote" button on products
  2. Configure Options - Select variants and optional products if available
  3. Review Quote Cart - View and modify items in the dedicated quote cart
  4. Submit Request - Customer submits the quote request with their details
  5. Receive Quote - Sales team receives the request and sends a formal quotation
